HTML: Tables

Use tables to display tabular data. Remember to label columns with a table header row.

Table Elements & Attributes
Name Syntax (as text) What it does Example
table <table> ... </table> Container for tabular data. <table>...</table>
thead <thead> ... </thead> Groups the header row(s) of a table. <thead><tr>...</tr></thead>
tbody <tbody> ... </tbody> Groups the main body rows. <tbody><tr>...</tr></tbody>
tr <tr> ... </tr> Defines a table row. <tr><td>A1</td><td>B1</td></tr>
th <th scope="col">Name</th> Header cell (use scope for accessibility). <th scope="row">Total</th>
td <td>Data</td> Standard table cell. <td>42</td>
caption <caption>Title</caption> Accessible title/label for the table. <caption>Monthly Sales</caption>

CSS: Selectors & Specificity

Common Selectors
Selector Pattern (as text) Matches Example
Type p All <p> elements p { margin: 0; }
Class .card Elements with class="card" .card { padding: 1rem; }
ID #main Element with id="main" #main { max-width: 960px; }
Descendant nav a <a> inside <nav> nav a { text-decoration: none; }
Attribute img[alt] <img> with an alt attribute img[alt] { border: 0; }
Pseudo-class a:hover Links on hover a:hover { opacity: .8; }
